Introduction

I've put this together as a starting point for development. You won't find any pretty design but the functionailty should be working ready for you to pull into your project. It's designed to be as light as possible, progressively enhanced and all that good stuff.

Usage

  1. Make sure you have a div#page-wrapper around all your content as this is what shifts over when the menu is activated. Generally this would be best as the first element directly after
  2. Enclose your menu in nav.main-menu. This is what will be pushed off screen using some absolute positioning*.

* The absolute positioning allows for a height of 100% making for a full height background to the menu area. Be careful declaring position on any of .main-menu's parent elements as this will affect it's positioning. position: relative; is set on #page-wrapper.
